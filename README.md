# Aktualizr-nano client for MCUXpresso SDK

PoC code for a Foundries.io OTA client (Aktualizr-nano) running on top of MCUXpresso SDK

## Prerequisites

- MCUXpresso SDK 2.11.0 RC1 for EVK-MIMXRT1064

- MCUBoot has to be loaded in the board (*SDK_2_11_0_EVK-MIMXRT1064/boards/evkmimxrt1064/mcuboot_opensource*)

- gcc-arm-none-eabi-10.3-2021.10

- cmake

## Recomended source code location

Create a new directory *SDK_2_11_0_EVK-MIMXRT1064/boards/evkmimxrt1064/fio_examples/*, and clone the repository inside it,
using the default *dm-ak-nano-mcuxpresso* subdirectory name.


## Building, signing and flashing
~~~
export PATH=$PATH:~/toolchains/gcc-arm-none-eabi-10.3-2021.10/bin
export ARMGCC_DIR=~/toolchains/gcc-arm-none-eabi-10.3-2021.10

cd armgcc
./build_flexspi_nor_debug.sh

python3 ../../../../../middleware/mcuboot_opensource/scripts/imgtool.py sign \
  --key ../../../../../middleware/mcuboot_opensource/root-rsa-2048.pem  \
  --align 4 \
  --header-size 0x400 \
  --pad-header \
  --slot-size 0x200000 \
  --version 2.11.0+1000 \
  flexspi_nor_debug/ota_demo.bin flexspi_nor_debug/ota_demo.1000.signed.bin

pyocd flash flexspi_nor_debug/ota_demo.1000.signed.bin -a 0x70040000
~~~
